# APP_INDICADORES_EDUCACION
La app busca visibilizar cómo se desempeña el sistema educativo argentino a través de cuatro indicadores comúnmente conocidos como de “eficiencia interna”: promoción efectiva, repitencia, abandono interanual y sobreedad. Permite explorar datos por provincia, nivel educativo y año, de manera clara y visual. 
El objetivo es facilitar la comprensión de las trayectorias escolares en todo el país, con un foco especial en Mendoza como caso de análisis. La app fue pensada tanto para investigadores y responsables de políticas públicas como para cualquier persona interesada en la temática. 
Los datos provienen del Relevamiento Anual elaborado por el Ministerio de Capital Humano de la Nación.
